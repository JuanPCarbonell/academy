# Day 3 — Tests, Documentation & Seeds

**Estimated time: ~7 hours (full day)**

Today you make your data trustworthy. By the end of the day you will have a full test suite, auto-generated documentation, and a seed table ready for tomorrow's snapshots exercise.

## Learning objectives
- Write and run generic tests (unique, not_null, accepted_values, relationships)
- Write custom singular tests for business logic validation
- Add descriptions to all models and columns using doc blocks
- Configure and load a seed table
- Master `dbt build` and node selection syntax

|---|---|---|
